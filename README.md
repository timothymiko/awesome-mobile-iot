[![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

# About
A collection of resources for Internet of Things mobile apps. The goal of this project is to help people build better mobile apps that interact with connected hardware such as wearables, smart home devices, etc.
